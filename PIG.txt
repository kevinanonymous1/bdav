Steps in Pig

1) Go to Local Terminal
2) create a txt file with Data:-
1,2,3
2,3,4
1,2,5
3,4,5
5,6,7
3,2,2
3)create Directory in Hadoop
- Hadoop fs -mkdir /pinput
- Hadoop fs -put pigdata.txt /pinput
- Hadoop fs- ls /pinput
4)Grunt > dat = Load '/pinput/pigdata.txt' using pigstorage(',') as (a1:int,a2:int,a3:int);
5)Grunt > dump dat ;

Q1) Display distinct  data:-

result = DISTINCT dat;
